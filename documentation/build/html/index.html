Welcome to RAT’s documentation!

Reflectivity Algorithms Toolbox (RAT)

RAT is a software package for the analysis of Reflectivity data. It is a combination of C/C++, MATLAB and Java that provides high performance, multi-threaded analysis. To know more about RAT and its history, please check the About RAT page.
