0004     % This makes the rosolutions class.
0005     % Backgrounds are defined in a two stage process. Firstly we define the
0006     % actual fitted parameters. These are held in a 'ParametersClass'
0007     % table. Then, we group these into the resolutions themselves using a
0008     % multiTypeTable. So, we can then use th eresolution parameters to
