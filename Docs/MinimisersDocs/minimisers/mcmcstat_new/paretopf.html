paretopf

PURPOSE

PARETOPF(x,a,b) Pareto density ba^b/x^(b+1)

SYNOPSIS

function y=paretopf(x,a,b)

DESCRIPTION 

 PARETOPF(x,a,b)  Pareto density ba^b/x^(b+1) 
 0 < a <= x,  b > 0
 mean(x) = a*b/(b-1)
 var(x) = a^2*b/(b-2)/(b-1)^2
