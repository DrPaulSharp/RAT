iact

PURPOSE

IACT estimates the integrated autocorrelation time

SYNOPSIS

function [tau,m] = iact(dati)

DESCRIPTION 

IACT estimates the integrated autocorrelation time
   using Sokal's adaptive truncated periodogram estimator.
