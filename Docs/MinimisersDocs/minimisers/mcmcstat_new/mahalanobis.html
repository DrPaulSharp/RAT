PURPOSE

MAHALANOBIS Malalanobis distance of rows in x

SYNOPSIS

function d=mahalanobis(x,mu,cmat,inverted)

DESCRIPTION 

MAHALANOBIS Malalanobis distance of rows in x
  mahalanobis(x,mu,cmat) calculates (x-mu)*inv(cmat)*(x-mu)'
  mahalanobis(x,mu,icmat,1) assumes cmat is inverted
