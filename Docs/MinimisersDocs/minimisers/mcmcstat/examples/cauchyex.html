0004 %% MCMC toolbox example
0005 % This example targets 10 dimensional Cauchy distribution. Cauchy
0006 % distribution does not have finite second moment, so the AM method,
0007 % which calculates chain variance will have troubles. In practice, it
0008 % gives good results. Here, we use RAM adaptation, which does not use
0009 % variance information (http://dx.doi.org/10.1007/s11222-011-9269-5).

0011 clear model options params

0014 npar = 10;

0016 model.ssfun    = @(x,d) 2*sum(log(1+x.^2));
0017 options.nsimu  = nsimu;
0018 options.method = 'ram';
0019 for i=1:npar, params{i} = {sprintf('x_{%d}',i), 0}; end
0020 
0021 [res,chain] = mcmcrun(model,[],params,options);

0025 iii = 1:min(npar,4); % plot first 4 chain columns
