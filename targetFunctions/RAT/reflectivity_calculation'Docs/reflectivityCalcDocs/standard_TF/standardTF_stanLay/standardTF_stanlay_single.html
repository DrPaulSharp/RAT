standardTF_stanlay_single

PURPOSE

Single threaded version of the standard layers, standardTF reflectivity

SYNOPSIS

function [outSsubs,backgs,qshifts,sfs,nbas,nbss,resols,chis,reflectivity,Simulation,shifted_data,layerSlds,sldProfiles,allLayers,allRoughs] = standardTF_stanlay_single(problemDef,problemDef_cells,problemDef_limits,controls)

DESCRIPTION 

 Single threaded version of the standard layers, standardTF reflectivity
 calculation. The function extracts the relevant parameters from the input
 arrays, allocates these on a pre-contrast basis, then calls the 'core' 
 calculation (the core layers standardTf calc is shared between multiple
 calculation types.
